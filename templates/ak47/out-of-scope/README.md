# Out-of-Scope Knowledge Base

此目录持久化记录被拒绝的功能请求。

**用途**:
- 避免重复讨论被拒功能
- 帮助 AI 保持决策一致性
- 记录决策理由和替代方案

**何时创建**:
- 明确拒绝的功能请求
- 与项目定位不符的需求
- 会破坏架构的决定

**格式**: 见 [out-of-scope-template.md](../templates/out-of-scope-template.md)

**查询方法**:
```bash
# 查看所有被拒功能
ls .ak47/out-of-scope/

# 搜索相关记录
grep -r "关键词" .ak47/out-of-scope/
```
